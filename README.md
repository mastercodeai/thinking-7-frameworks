# thinking-7-frameworks

> 华尔街和硅谷大佬用来推动数十亿美元决策的 7 个顶级思维框架

覆盖商业决策、项目立项、人生转折全场景，7 个框架互相交叉验证，形成完整决策系统。

## 7 个框架

| # | 框架 | 来源 | 解决什么问题 |
|---|------|------|-------------|
| 1 | **第一性原理** | 伊隆·马斯克 | 你的假设是真的吗？ |
| 2 | **逆向思维** | 查理·芒格 | 什么会导致失败？ |
| 3 | **5 个为什么** | 丰田生产系统 | 根本原因是什么？ |
| 4 | **二阶思维** | 霍华德·马克斯 | 下一步会发生什么？ |
| 5 | **后悔最小化** | 杰夫·贝索斯 | 80 岁的你会后悔什么？ |
| 6 | **机会成本** | 经济学基本原理 | 你在放弃什么？ |
| 7 | **事前验尸** | 加里·克莱因 | 什么会杀死这个项目？ |

## 交叉验证网络

```
第一性原理 × 逆向思维：基本事实 vs 失败模式
5个为什么 × 事前验尸：根因 vs 死因
二阶思维 × 机会成本：连锁反应 vs 放弃的选项
后悔最小化 × 所有框架：最终情感检验
```

## 使用

```
# 单框架分析
用第一性原理帮我拆解这个问题

# 决策矩阵（推荐，自动选 3-4 个框架）
我该不该从大厂辞职去创业？

# 全框架扫描
帮我做一个完整的决策分析
```

## 使用建议

1. 提供充分上下文：你的决定是什么？有哪些选项？你最担心什么？
2. 诚实回答：框架的有效性取决于你提供的信息是否真实
3. 不要跳过交叉验证：单个框架的结论可能是错的，多个框架一致的结论才可靠
4. 重大决策用全框架扫描：职业转换、创业、大额投资——值得花 30 分钟用全部 7 个框架
